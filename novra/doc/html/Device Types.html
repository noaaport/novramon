Novra Device Types

S75 Version 2.x

A satellite receiver with the following capabilities ...
  • performs MPE processing on PIDs that contain IP traffic
  • can forward raw traffic to a given IP address and UDP port
  • supports processing of 32 simultaneous PIDs
  • broadcasts status packets on a subnet

S75 Version 3

A satellite receiver with the following capabilities ...

  • performs MPE processing on PIDs that contain IP traffic
  • supports processing of 16 simultabeous PIDs
  • supports IGMP filtering for multicast traffic
  • broadcasts status packets on a subnet
  • can unicast status packets 
  • provides a long line compensation switch for longer coaxial cables
  • provides a high voltage switch to increase the signal to noise ratio of the LNB signal
  • allows the DVB MAC address to be configured

S75 Version 5 (PesMaker)

A satellite receiver with the following capabilities ...
  • performs MPE processing on PIDs that contain IP traffic
  • can forward individual raw traffic PIDs to seperate IP addresses and UDP ports
  • can forward PES packets to seperate IP addresses and UDP ports
  • supports processing of 32 simultaneous PIDs
  • supports IGMP filtering for multicast traffic
  • broadcasts status packets on a subnet
  • can unicast status packets 
  • provides a long line compensation switch for longer coaxial cables
  • provides a high voltage switch to increase the signal to noise ratio of the LNB signal

S75+ (S75 V3 functional replacement)

A satellite receiver with the following capabilities ...
  • performs MPE processing on PIDs that contain IP traffic
  • supports processing of 32 simultaneous PIDs
  • supports IGMP filtering for multicast traffic
  • broadcasts status packets on a subnet
  • can unicast status packets 
  • can be polled for status
  • allows for selection LNB control parameters (tone frequency and switching voltages)
  • provides a long line compensation switch for longer coaxial cables
  • provides a high voltage switch to increase the signal to noise ratio of the LNB signal

S75 Conditional Access (S75CA)

A satellite receiver with the following capabilities ...
  • has conditional access module support for descrambling DVB programs
  • performs MPE processing on PIDs that contain IP traffic
  • supports processing of 32 simultaneous PIDs
  • supports IGMP filtering for multicast traffic
  • broadcasts status packets on a subnet
  • can unicast status packets 
  • can be polled for status
  • allows for selection LNB control parameters (tone frequency and switching voltages)
  • provides a long line compensation switch for longer coaxial cables
  • provides a high voltage switch to increase the signal to noise ratio of the LNB signal

S75 Fixed Key (S75FK)

A satellite receiver with the following capabilities ...
  • has fixed key support for descrambling DVB PIDs
  • performs MPE processing on PIDs that contain IP traffic
  • supports processing of 32 simultaneous PIDs
  • supports IGMP filtering for multicast traffic
  • broadcasts status packets on a subnet
  • can unicast status packets 
  • can be polled for status
  • allows for selection LNB control parameters (tone frequency and switching voltages)
  • provides a long line compensation switch for longer coaxial cables
  • provides a high voltage switch to increase the signal to noise ratio of the LNB signal

S75+ Pro, S75CA Pro and S75FK Pro

Satellite receivers as described above with added support for the Pro feature set. The Pro feature set includes ...
  • can forward individual raw traffic PIDs to multipled IP addresses and UDP ports
  • can forward an entire transport stream (with or without NULL packets) to an assigned IP address and UDP port
  • can generate program association tables for chosen DVB programs (to allow viewing of video by programs like VLC)
