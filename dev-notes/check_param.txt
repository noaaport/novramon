In the s200 and s300 the is no "signal strength as percentage". We could
calculate ir using (borrowed from the library files)

if(ss_dbm > -25){
  /* ss_per = 101; */
  ss_per = 100;
}else{
  ss_per = 2 * ( ss_dbm + 75 );
}

We implement this in the function fill_novra_param() in status.cc.
